# Tetros

A **Tetris-inspired puzzle game** built with C# and MonoGame (XNA Framework). Test your skills by stacking falling bricks, completing rows, and achieving the highest score!

## Features

- 🎮 **Classic Tetris Gameplay** - Familiar mechanics with smooth controls
- 🧱 **Multiple Brick Types** - Various tetromino shapes with proper rotation
- 📊 **Grid-Based Field** - 10×16 playing field with collision detection
- ⚡ **Gravity System** - Realistic brick falling mechanics
- 🎨 **Colorful Visuals** - Distinct textures for different brick types
- 🔧 **Time-Based Events** - Efficient event management system

## Technology Stack

- **Language**: C# (.NET 8)
- **Framework**: MonoGame (XNA Framework)
- **Graphics**: SpriteBatch rendering

## How to Play

- **← / →** - Move brick left/right
- **↓** - Accelerate brick fall
- **Z / X** - Rotate brick
- **Complete rows** to clear them and earn points

## Project Structure

- `gameParts/` - Core game mechanics and brick definitions
- `Bricks.cs` - Brick classes and factory pattern implementation
